Annotated screenshots of the eResources service for use in the project README, blog posts, and conference materials.

## Conventions

- File names: `kebab-case-noun.png`. URL-safe — no spaces, no caps.
- Prefer PNG for UI screenshots; JPG for photos.
- **No patron data, no real card numbers, no real email addresses** — use staff test accounts or scrub before saving.
- If a screenshot shows the staff admin UI, make sure no real OAuth account name is visible (sign in with a generic placeholder Workspace account, or crop the avatar/menu out).
